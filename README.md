# Athena Website

Marketing landing page with dummy Login / Sign Up pages.

## Quick Start

```bash
cd client && npm install && npm run dev
```

Client runs on [http://localhost:5174](http://localhost:5174).

## Stack

- **Client**: React 19, React Router, Vite, Tailwind CSS v4, Motion, Lucide React
- No backend required — fully static site
